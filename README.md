## Java Lab Practices Repository

This repository contains the Java code developed during lab sessions of the Java II (CMPE_114) course at TED University.

The repository focuses on Object-Oriented Programming (OOP) concepts, demonstrating the four pillars of OOP through practical implementations.

Implemented topics and projects include:
- Basic class inheritance demonstration
- Library Management System
- Employee Management System
- Shop Price Management System

Language: 
- Java
